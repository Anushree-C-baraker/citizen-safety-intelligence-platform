# Citizen Safety Intelligence Platform

## Description
An AI-powered web application that helps users assess safety risks based on location, weather, crowd density, lighting conditions, and crime rates.

## Features
- Location Dashboard
- Safety Prediction
- Safety Hotspots Map
- Community Alerts
- Emergency Assistance
- Daily Safety Tips

## Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-Learn
- Folium
- Plotly

## Deployment
Streamlit Community Cloud
